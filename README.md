These are my personal Arch dotfiles.

### Setup Instructions

1. Run setup.sh as root
```sh
sudo setup.sh
```
2. Change shell
```sh
chsh -s /usr/bin/zsh
```
3. Reboot
```sh
sudo reboot
```
4. Clone dotfiles in ~
```sh
git clone https://github.com/sebastian-j-ibanez/dotfiles.git
```
4. Run stow.sh
```sh
stow.sh
```
