# Dotfiles
These are my personal CachyOS dotfiles.

### Setup Instructions

1. Clone repository
```sh
git clone https://github.com/sebastian-j-ibanez/dotfiles.git && cd dotfiles/
```
2. Run install.sh
```sh
./install.sh
```
3. Change shell
```sh
chsh -s /usr/bin/zsh
```
4. Reboot
```sh
sudo reboot
```
5. Run stow.sh
```sh
stow.sh
```
